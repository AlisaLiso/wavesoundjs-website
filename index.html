wavesoundjs

v1.0.3

TypeScript component for interacting with audio via audio wave built on Canvas.

Checkout the code you can at github github and for more information on package go to npm npm

npm install wavesoundjs

Developed with jekyll by alisaliso alisaliso

Getting started

Install wavesoundjs npm package:

$ npm install wavesoundjs

Create a container where you want the waveform to appear:

<div id="wavesoundjs"></div>

Follow along https://github.com/bbc/audiowaveform to install a C++ command-line application that generates waveform data

Next, in your JavaScript code, create an instance of the global WaveSurfer object. 


              const container = document.querySelector(‘#wavesoundjs’); let
              wavesoundjs = wavesoundjs.create({ container, points: [-1, -3...]
              });

Parameters

Wavesoundjs requires to parameters. Unique DOM element and array of peaks that was generated with audiowaveform -> https://github.com/bbc/audiowaveform and settings.

Generate peaks with installed audiowaveform in terminal

$ audiowaveform -i balladtempest_fields_lkp_64kb.mp3 --pixels-per-second 4 -b 8 -o some-file.json
